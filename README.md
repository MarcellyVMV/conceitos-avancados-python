## Sobre a disciplina

A disciplina teve como foco o desenvolvimento de software utilizando conceitos avançados de programação em Python, com ênfase em programação orientada a objetos, organização de código, reutilização de componentes e aplicação de bibliotecas científicas.

Ao longo do semestre, foram explorados conceitos fundamentais para modelagem de sistemas, construção de interfaces gráficas, persistência de dados e desenvolvimento de aplicações mais robustas e escaláveis.

### Ementa

- Projeto e construção de programas utilizando modelagem orientada a objetos;
- Tratamento de exceções;
- Organização, reutilização e manutenção de código;
- Aplicação de boas práticas de programação;
- Desenvolvimento de interfaces gráficas para interação com o usuário;
- Persistência e manipulação de dados;
- Utilização de pacotes científicos, matemáticos e computacionais para resolução de problemas.

### Conteúdos abordados

Durante a disciplina, foram estudados tópicos como:

- Classes e objetos
- Encapsulamento
- Composição
- Herança
- Polimorfismo
- Métodos especiais
- Tratamento de exceções
- Modularização e organização de projetos
- Interfaces gráficas com PyQt
- Persistência de dados
- Computação científica com NumPy
- Métodos científicos com SciPy
- Visualização de dados com Matplotlib
